# Controls  

## 1st-Person  
w / a / s / d - walk around  
mouse - look around  
space - enable cursor mode  
tab - enable 3rd-Person mode

## 3rd-Person Debug Mode
w / a / s / d - walk around  
mouse - look around  
space - enable cursor mode  
tab - enable 1st-Person mode

## Cursor Mode
space - disable cursor mode  
File > Load.. - Load different world  
View > Debug Output - Display debug information
